# 考研学习进度追踪 (Study Tracker)

这是一个专门为考研学习设计的进度追踪与实时互动应用。

## 核心功能
1. **专注计时**：开始学习时启动计时器，支持记录各科目学习时长。
2. **每日总结与打分**：每天学习结束后，可以对当天的状态进行 1-5 星打分并撰写小结。
3. **实时状态看板**：通过另一台设备实时查看当前的专注状态（开发中...）。
4. **奖励兑换所**：专注时长积累积分，可用于兑换奖励。

## 技术架构
- **纯前端实现**：无需复杂的编译环境 (No Node.js needed)，采用最纯粹的 HTML5 + 原生 CSS + Vanilla JavaScript 构建。
- **UI 风格**：全屏自适应、暗黑模式、毛玻璃 (Glassmorphism) 拟物化设计，为 iPad 用户提供最高质量的情绪价值与视觉体验。

## 运行方式
你可以直接双击 `index.html` 在浏览器中打开，或者如果安装了 Python，可以使用以下命令启动本地服务器：

```bash
python -m http.server 8000
```
然后在浏览器访问 `http://localhost:8000` 即可体验。
